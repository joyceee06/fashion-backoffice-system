# 👗 Fashion Back-Office System

A web-based back-office system developed to manage fashion products, customers, staff, and orders in a retail environment.

---

## 📌 Overview

This project is a full-stack web application built using PHP and MySQL. It provides administrative functionalities for managing daily business operations such as inventory, order processing, and user access control.

---

## 🚀 Features

* 🔹 CRUD operations for products, customers, staff, and orders
* 🔹 User authentication system with role-based access (Admin & Non-admin)
* 🔹 Secure password handling using hashing
* 🔹 Responsive user interface using Bootstrap
* 🔹 Interactive product table with search, pagination, and sorting (DataTables)
* 🔹 Product details displayed using Bootstrap modal
* 🔹 Invoice generation for order management

---

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML, CSS
* Bootstrap
* JavaScript
* jQuery (DataTables)

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/fashion-backoffice-system.git
```

### 2. Move project to XAMPP

Place the project folder inside:

```
htdocs/
```

### 3. Configure database

* Open `database.php`
* Update with your MySQL credentials

### 4. Import database

* Open phpMyAdmin
* Create a database
* Import the provided `.sql` file

### 5. Run the project

Open your browser:

```
http://localhost/fashion-backoffice-system
```
